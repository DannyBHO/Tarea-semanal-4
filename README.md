Este ejemplo modela un sistema básico de reservas de hotel utilizando
Programación Orientada a Objetos

Se representan entidades del mundo real como el Hotel, las Habitaciones y las Reservas mediante
clases con atributos y métodos, permitiendo simular la interacción entre los objetos.

Se aplican los principios de POO:
 Encapsulación
 Abstracción
 Uso de clases y objetos
 Interacción entre objetos
